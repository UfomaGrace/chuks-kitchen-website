# Chuks Kitchen

A modern and responsive food ordering web application built with React and Tailwind CSS.
This project showcases a clean UI for browsing Nigerian home-cooked meals, viewing menus, and exploring featured dishes.

---

* Beautiful onboarding page with hero image

## Installation

1. Clone the repository
   git clone https://github.com/UfomaGrace/chuks-kitchen-website

2. Navigate into the project
   cd chuks-kitchen

3. Install dependencies
   npm install

4. Start the development server
   npm run dev
